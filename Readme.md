# Smart Calendar System

An AI-powered calendar that optimizes your schedule using advanced algorithms.

## Features

- Multiple scheduling algorithms (Branch & Bound, Hungarian, Genetic)
- Google Calendar integration
- Streamlit web interface
- Preference-based optimization
